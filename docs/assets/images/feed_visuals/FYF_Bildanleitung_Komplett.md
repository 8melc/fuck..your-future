## Stilrichtlinien für die Bildauswahl

Bei der Auswahl der Bilder auf Unsplash achten Sie auf:

✓ **Minimal:** Wenige Elemente, viel Raum, klare Komposition  
✓ **Ehrlich:** Keine überbearbeiteten Hochglanzbilder  
✓ **Ruhig:** Gedämpfte Farben, keine grellen Kontraste  
✓ **Schwarz/Weiß oder Monochrom:** Bevorzugt, aber nicht zwingend  
✓ **Natürliches Licht:** Keine künstliche Studio-Beleuchtung  
✓ **Abstrakt/Konzeptionell:** Symbolische Darstellung der Visual-Idee  

✗ **Vermeiden:** Stock-Photo-Ästhetik, Menschen im Fokus, bunte Farben, Perfektion

## Batch-Download Script (optional)

Falls Sie die Bilder automatisch herunterladen möchten, können Sie dieses Python-Script verwenden:

```python
import requests
import time

images = [
    ("https://source.unsplash.com/1200x900/?sand,wood,minimal", "fyf_zeit_endlichkeit_seneca_ueber_die.jpg"),
    # ... weitere URLs
]

for url, filename in images:
    response = requests.get(url, allow_redirects=True)
    with open(filename, 'wb') as f:
        f.write(response.content)
    time.sleep(2)  # Rate limiting
    print(f"✓ {filename} heruntergeladen")
```

**Hinweis:** Dieses Script benötigt die `requests` Library (`pip install requests`)
